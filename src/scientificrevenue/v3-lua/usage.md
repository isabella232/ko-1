### Initialize Plugin
```lua
sdkbox.ScientificRevenue:init()
```

### Set Pricing Session Options
```lua
sdkbox.ScientificRevenue:sessionOptions(true, "en_US", true)
```

### Start Session
```lua
sdkbox.ScientificRevenue:startSession("UserName");
```
